#### 绑定域名

#### 添加菜单页

添加菜单页的思路（添加菜单页就是添加一个页面，有两种方式）：第一种是使用命令 `hexo new page "photo"` 就直接创建了 `blog\source\photo\index.md` 文件，然后编辑 `index.md` 文件就可以了。

第二种：手动创建上面的文件

- 在主题的配置文件添加 `menu` 索引路径（根路径是 `blog\source`，所以你如果想要更改页面的内容就去 `hexo\source` 下找到对应的文件夹，默认内容是在其 `index.md` 文件下)
- 在 blog 的 source 文件下添加对应的文件夹
- 在主题的配置文件添加menu_icon字段设置对应的icon
- 修改language文件下zh-Hans语言包
- 在发表文章的时候添加对应的menu字段就可以看到

> 比如，我要添加一个名为"导航"的菜单页

修改主题配置文件中的内容，增加导航菜单，并为其指定图标。图标可以去 [FontAwsome](http://fontawesome.io/)，然后鼠标往下拉，在图标集中选择自己喜欢的 icon，然后记住名字，保存在上面的 menu_icon 字段中就可以啦。

```yml
menu:
  # home: /
  navigation: /navigation/
  categories: /categories/
  about: /about/
  # archives: /archives/
  tags: /tags/
  #sitemap: /sitemap.xml
  #commonweal: /404/

menu_icons:
  enable: true
  #KeyMapsToMenuItemKey: NameOfTheIconFromFontAwesome
  home: home
  about: user
  categories: th
  schedule: calendar
  tags: tags
  archives: archive
  sitemap: sitemap
  commonweal: heartbeat
  navigation: icon-bookmark
```

修改 `next/language/zh-Hans.yml`，添加翻译字段：

```
menu:
  home: 首页
  archives: 归档
  categories: 分类
  tags: 标签
  about: 关于
  search: 搜索
  schedule: 日程表
  sitemap: 站点地图
  commonweal: 公益404
  navigation: 导航
```

新建菜单页，并发布：

```
$ hexo new page "navigation"
$ hexo clean
$ hexo generate
$ hexo deploy
```

以上，便完成了新菜单也“导航”的添加。

#### 添加近期文章模块

将此代码贴在 `next/layout/_macro/sidebar.swig` 中的 `if theme.links` 对应的 `endif` 后面，就ok了，是不是很简单。

```swig
{% if theme.recent_posts %}
    <div class="links-of-blogroll motion-element {{ "links-of-blogroll-" + theme.recent_posts_layout  }}">
      <div class="links-of-blogroll-title">
        <!-- modify icon to fire by szw -->
        <i class="fa fa-history fa-{{ theme.recent_posts_icon | lower }}" aria-hidden="true"></i>
        {{ theme.recent_posts_title }}
      </div>
      <ul class="links-of-blogroll-list">
        {% set posts = site.posts.sort('-date') %}
        {% for post in posts.slice('0', '5') %}
          <li>
            <a href="{{ url_for(post.path) }}" title="{{ post.title }}" target="_blank">{{ post.title }}</a>
          </li>
        {% endfor %}
      </ul>
    </div>
{% endif %}
```

为了配置方便，在主题的 `_config.yml` 中添加了几个变量，如下：

```yml
recent_posts_title: 近期文章
recent_posts_layout: block
recent_posts: true
```

#### 增加友情链接功能

在主题配置文件中找到 `links` 属性，修改 `links_title` 属性的值为“友情链接”（也可以是其他文字），然后添加上好友的博客名称和博客地址，其格式如下：

```yml
# Blog rolls
links_title: 友情链接
#links_layout: block
links_layout: inline
links:
  seyvoue: https://www.seyvoue.com/

```

#### 增加文末评论默认收起的功能

实现当打开一篇文章，评论默认关闭，需要手动点击打开评论，文末的评论才打开的功能：

修改 `next/layout/partials/comments.swig`，找到下面的内容：文件中的内容：

```swig
{% elseif theme.vkontakte_api.enable and theme.vkontakte_api.comments %}
  <div id="vk_comments"></div>
{% elseif theme.disqus.enable %}
```

在其后增加以下内容：

```swig
  <div id="disqus_switch" class="center">
    <a href="javascript:loadDisqus();">
      显示评论，需翻墙<br/>Show Comments
    </a>
  </div>
  <script type="text/javascript">
    function loadDisqus() {
      $('#disqus_switch').hide();
      $('#disqus_thread').show();
      run_disqus_script('count.js');
      run_disqus_script('embed.js');
    }
  </script>
```

安装插件：

```shell
$ npm install hexo-generator-feed --save
```

在站点目录下的 `_config.yml` 中，增加以下内容：

```yml
#RSS 订阅插件
plugin:
- hexo-generator-feed

#RSS 插件配置
feed:
  type: rss2
  path: rss.xml
  limit: 20
  hub:
  content: true
```
